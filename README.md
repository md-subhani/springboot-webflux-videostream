# springbott-webflux-videostream
This is sample reference for video streaming by using springboot and web flux

Data Base : MongoDb

MongoDb set up: 
1. installed mongo db in ur local
2. go to the bin folder of mongodb installed location
3. open command prompt at this location example (C:\....\mongodb\mongodb-win32-x86_64-windows-8.2.6\bin)
4. run mongod command

Clone the project in ur workspace
1.mvn clean install
2.Run as Spring boot application

Default port code of Mongo db 27017
install mongoDBCompass
